Mini Weather Widget

Local Weather

Enable Geolocations to get current weather


Doing this project gave me my first experiences with a open web API. I used api.openweathermap.org in coordination with the browsers geolocation ability to get some local weather data in JSON format. As I started to tinker with the API, I quickly leanred that I needed to start using "sessionStorage" to keep variable in memory or I would hit my API call limit very quick and be locked out from the servers.

A very rewarding little project that wound up teaching quite a few lessons.
